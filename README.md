# Android-JAVA-Regex-Pattern
- This is a regular expression available on Android.

## How to use Getter
```
    YHPattern.IPv4("0.0.0.0"); // true
    YHPattern.IPv4("255.255.255.255"); // true
    YHPattern.IPv4("0.0.0.256"); // false
```
